# Setup Checklist — Run Your First Nakhla Session in 1 Week

Step-by-step for a teacher to go from zero to Session 1.

---

## Day 1: Google Classroom + Zoom Setup

### Google Classroom
- [ ] Confirm you have a Google Workspace for Education account (free for schools)
- [ ] If not, apply at edu.google.com — approval usually takes 1–3 days
- [ ] Create two classrooms:
  - **"Nakhla — Israel Class"**
  - **"Nakhla — UAE Class"**
- [ ] Invite your co-teacher from the partner school as a co-teacher in both
- [ ] Create Topics in each classroom: "Session 1", "Session 2", "Session 3"
- [ ] Share join codes with partner teacher

### Zoom
- [ ] Create a Zoom account (or confirm school Zoom Pro license — free tier has 40-min limit)
- [ ] Schedule a **recurring meeting** for all 3 session dates
- [ ] Enable **Language Interpretation:**
  - Settings → In Meeting (Advanced) → Language Interpretation → Toggle ON
  - When scheduling, click Interpretation → add Hebrew, Arabic, English channels
- [ ] Enable **Breakout Rooms:**
  - Settings → In Meeting (Advanced) → Breakout Room → Toggle ON
  - Check "Allow host to assign participants to breakout rooms when scheduling"
- [ ] Save the meeting link — share it in both Google Classrooms

---

## Day 2: Blooket + Padlet Accounts

### Blooket
- [ ] Create a free teacher account at blooket.com
- [ ] Create a Question Set for Session 1 using `content/session-01-greetings/kahoot-questions.json`
  - See `content/session-01-greetings/blooket-setup.md` for step-by-step
- [ ] Test: host a game and join as a student in another browser tab
- [ ] Repeat for Session 2 and 3 question sets (or do later)

### Padlet
- [ ] Create a free account at padlet.com (free tier = 3 boards — one per session)
- [ ] Create board: **"Nakhla — Hello Cards"** (Wall layout)
  - Set to "Can write" for anyone with link
  - Enable **"Require approval"** (Moderated mode — teacher approves posts)
  - Enable content filtering
  - Set attribution to first name only
- [ ] Create a sample post showing the Hello Card format (see `content/session-01-greetings/activity-padlet.md`)
- [ ] Test: open the Padlet link in an incognito window to verify students can post without login

---

## Day 3: Microsoft Flip Group + Parent Invitation

### Microsoft Flip
- [ ] Go to flip.com and sign in with your Microsoft or Google educator account (free)
- [ ] Create a Group: **"Nakhla Exchange"**
- [ ] Create Topic 1: **"Hello! Introduce Yourself"**
  - Video length: 30–60 seconds
  - Moderation: ON
  - See `content/session-01-greetings/flip-prompt.md` for the exact prompt
- [ ] Share the Group join code in Google Classroom
- [ ] Test: record a sample video yourself to verify the workflow

### Parent Communication
- [ ] Create a WhatsApp group: **"Nakhla Parents — [School Name]"**
- [ ] Set to **admin-only messaging** (Settings → Group Settings → Send Messages → Only Admins)
- [ ] Add all parents from your class
- [ ] Send the welcome message (template in `program/PLATFORMS.md` → WhatsApp section)
- [ ] Include: program overview, session dates, what to expect, parent consent info
- [ ] Mention Microsoft Flip — parents can view their child's approved videos

---

## Day 4: YouTube Playlist Creation

### YouTube Unlisted Playlists
- [ ] Sign in to YouTube with your school Google account
- [ ] Create playlist: **"Nakhla — Session 1: Hello, Neighbor!"** (set to Unlisted)
- [ ] Add 3–5 short (2–5 min) age-appropriate videos about:
  - Greetings and daily life in Israel and the UAE
  - Kid-friendly cultural introductions
  - Geography — where are these countries?
- [ ] Share the playlist link in Google Classroom under Session 1 Topic
- [ ] Repeat for Session 2 and 3 playlists (or do later)

### Content Guidelines
- All videos must be age-appropriate (8–12 years old)
- No political or conflict-related content
- Focus on daily life, traditions, food, music, nature, geography
- Include videos from both Israeli and Emirati creators where possible

---

## Day 5: Dry Run (Internal)

### Technical Rehearsal (30 minutes, with moderators)
- [ ] Everyone joins the Zoom meeting
- [ ] Test: can teacher create breakout rooms?
- [ ] Test: Interpretation channels work — assign someone to Hebrew channel, someone to Arabic
- [ ] Test: moderators can join/leave breakout rooms
- [ ] Test: screen sharing works for Blooket
- [ ] Test: Padlet link opens and students can post without login
- [ ] Test: Blooket game code works — moderators join as participants
- [ ] Test: audio/video quality for all participants
- [ ] Test: Microsoft Flip — moderators record a test video
- [ ] Identify backup plan if Zoom fails (Google Meet or WhatsApp voice notes)

### Prepare Session 2 & 3 Materials (if not done)
- [ ] Create remaining Padlet boards (Session 2: Grid layout, Session 3: Wall layout)
- [ ] Create remaining Blooket Question Sets
- [ ] Create remaining Flip Topics
- [ ] Create remaining YouTube playlists

---

## Day 6–7: Recruit Pilot Classes + Send Parent Consent

### Recruit Moderators
- [ ] You need 1 adult moderator per 2–3 pods (e.g., 20 kids = 5 pods = 2 moderators)
- [ ] Options: teaching assistants, parent volunteers, student teachers, school counselors
- [ ] Share the Moderator Playbook (`program/MODERATOR-PLAYBOOK.md`) with each moderator
- [ ] Schedule a 15-minute briefing call with moderators

### Moderator Briefing Agenda (15 min)
1. What is Nakhla? (2 min)
2. Session format walkthrough (3 min)
3. Zoom breakout room + Interpretation mechanics (3 min)
4. Escalation scenarios — read through all 5 (5 min)
5. Questions (2 min)

### Student Prep
- [ ] Tell students about the program — show them the map (Israel + UAE)
- [ ] Teach them 3 words: Hello (שלום / مرحبا), Thank you (תודה / شكراً), Friend (חבר / صديق)
- [ ] Explain the format: "You'll meet kids from [country] and talk in small groups"
- [ ] Reassure: "You don't need to be perfect in any language. English is your bridge."
- [ ] Assign the Session 1 YouTube playlist as homework
- [ ] Assign the first Flip video: "Introduce Yourself" (see `content/session-01-greetings/flip-prompt.md`)

### Parent Consent
- [ ] Send parent consent form via Google Classroom or WhatsApp
- [ ] Include: program description, tools used, data handling, video recording policy
- [ ] Collect signed forms before Session 1
- [ ] Note: Microsoft Flip is COPPA/FERPA compliant — mention this in the consent form

### Partner School Coordination
- [ ] Confirm session date and time with partner teacher (account for timezone differences!)
  - Israel: UTC+2 (or UTC+3 during DST)
  - UAE: UTC+4
  - A session at 14:00 Israel time = 16:00 UAE time
- [ ] Share Zoom link, Google Classroom codes, Flip group code, and Padlet links
- [ ] Exchange student lists (first names only — no last names or photos shared externally)
- [ ] Agree on pod assignments: 2 Israeli + 2 Emirati per pod

---

## Session 1 Day!

### 30 Minutes Before
- [ ] Open Zoom, test audio/video
- [ ] Open Blooket dashboard, ready to host Session 1 game
- [ ] Open Padlet board in a separate tab
- [ ] Have discussion prompts printed or on a second screen
- [ ] Confirm moderators are online
- [ ] Remind students about the Interpretation globe icon in Zoom
- [ ] Text partner teacher: "Ready?"

### During Session
- [ ] Follow the Session 1 plan in `program/PROGRAM.md`
- [ ] Full Group → Pod Pairs (breakout rooms) → Activity → Wrap-Up
- [ ] Share Padlet link in Zoom chat for the Hello Card activity
- [ ] Share Blooket game code for the quiz

### After Session
- [ ] Send parent tip via WhatsApp (from `content/session-01-greetings/parent-tip.md`)
- [ ] Review and approve any pending Padlet posts
- [ ] Review and approve any pending Flip videos
- [ ] Share the Padlet board link and Flip topic link with parents
- [ ] Post reflection assignment in Google Classroom: "What new word did you learn today?"
- [ ] Debrief with co-teacher and moderators: what worked? What to adjust?
- [ ] Confirm date for Session 2
