# Activity: Our World Board — Collaborative Jamboard

## Overview
Each pod creates a shared board showing their daily lives, divided into four themes.

**Duration:** 15 minutes
**Group:** Pod pairs on shared Jamboard/Google Slides

---

## Setup (Teacher, Before Session)
1. Create one Jamboard (or Google Slides) per pod
2. Set sharing to "Anyone with the link can edit"
3. Prepare each board with a 4-square template:

### Template Layout
```
┌──────────────────────┬──────────────────────┐
│    🍽️ MY FAVORITE     │    👨‍👩‍👧‍👦 MY FAMILY      │
│       FOOD            │                      │
│                       │                      │
│  (Draw or describe)   │  (Draw or describe)  │
│                       │                      │
├──────────────────────┼──────────────────────┤
│    🎵 MY MUSIC        │    🎮 MY GAMES        │
│                       │                      │
│                       │                      │
│  (Draw or describe)   │  (Draw or describe)  │
│                       │                      │
└──────────────────────┴──────────────────────┘
```

Each kid picks 1–2 squares to fill.

---

## Instructions for Kids

**EN:**
1. Your pod has a board with 4 squares: Food, Family, Music, Games
2. Pick one or two squares to fill in
3. Draw a picture, write words, or add images from Google
4. Try to include a word in your language!
5. You have 15 minutes — work together!

**עב:**
1. לקבוצה שלכם יש לוח עם 4 ריבועים: אוכל, משפחה, מוזיקה, משחקים
2. בחרו ריבוע אחד או שניים למלא
3. ציירו תמונה, כתבו מילים, או הוסיפו תמונות מגוגל
4. נסו לכלול מילה בשפה שלכם!
5. יש לכם 15 דקות — עבדו יחד!

**عر:**
1. مجموعتكم لديها لوحة بأربعة مربعات: طعام، عائلة، موسيقى، ألعاب
2. اختاروا مربعاً أو اثنين لملئهما
3. ارسموا صورة، اكتبوا كلمات، أو أضيفوا صوراً من جوجل
4. حاولوا إضافة كلمة بلغتكم!
5. لديكم ١٥ دقيقة — اعملوا معاً!

---

## Sharing (Last 5 Minutes of Activity Block)
- Return to main room
- Teacher screen-shares each pod's board (30 seconds each)
- Teacher highlights similarities: "Both groups drew rice!" / "You both like football!"
- Celebrate the boards — these go on the Google Sites portfolio
