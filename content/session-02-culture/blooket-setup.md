# Blooket Setup — Session 2: Our World!

Step-by-step to create the Blooket game from `kahoot-questions.json`.

---

## Create the Question Set

1. Go to **blooket.com** and log in to your teacher account
2. Click **Create** → **Create a Set**
3. **Title:** "Nakhla Session 2: Our World!"
4. **Description:** "Trilingual quiz about food, family, music, and games"
5. Add each question from `kahoot-questions.json`:

### Questions to Add

| # | Question | Correct Answer |
|---|----------|---------------|
| 1 | How do you say 'Food' in Hebrew? | אֹכֶל (Okhel) |
| 2 | How do you say 'Family' in Arabic? | عَائِلَة (A'ila) |
| 3 | What does 'Ta'im' (טָעִים) mean? | Delicious |
| 4 | What does 'Ladhidh' (لَذِيذ) mean? | Delicious |
| 5 | Which food is popular in BOTH Israel and the UAE? | Hummus (חומוס / حُمُّص) |
| 6 | How do you say 'Mom' in Hebrew? | אִמָּא (Ima) |
| 7 | How do you say 'Brother' in Arabic? | أَخ (Akh) |
| 8 | What does 'Hayya nal'ab!' (هَيَّا نَلْعَب!) mean? | Let's play! |
| 9 | Which word sounds similar in both Hebrew and Arabic? | Muzika / Musiqa (Music) |
| 10 | The date palm grows in both Israel and the UAE. True or false? | True! |

6. For each question: enter the question text, add 4 answer options, and select the correct one
7. **Time limit:** 30 seconds per question
8. Click **Save**

---

## Host the Game

1. From your Blooket dashboard, find the Question Set
2. Click **Host**
3. Choose game mode: **Gold Quest** (recommended) or **Tower Defense** (good alternative)
4. Students go to **play.blooket.com** and enter the Game ID
5. Students enter a nickname (first name only) — no account needed
6. Share your screen via Zoom so everyone sees the game
7. After each question, read the correct answer aloud in all 3 languages

---

## Tips
- Session 2 questions highlight shared culture (hummus, date palms, similar words) — emphasize these moments!
- If a student shouts "We have that too!" — that's the magic. Celebrate it.
- Free tier: unlimited games, up to 60 players
