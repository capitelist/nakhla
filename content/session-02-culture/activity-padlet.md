# Activity: Our World Board — Collaborative Padlet

## Overview
Each pod creates posts on a shared Padlet board showing their daily lives across four themes.

**Duration:** 15 minutes
**Group:** Pod pairs posting to shared Padlet board
**Tool:** Padlet (padlet.com) — no student account needed

---

## Setup (Teacher, Before Session)
1. Create a Padlet board: **"Nakhla — Our World"** (Grid layout with 4 columns)
2. Set to "Can write" for anyone with link
3. Enable "Require approval" so teacher reviews posts before they appear
4. Create 4 column headers (sections):

### Board Layout
```
🍽️ MY FAVORITE FOOD  |  👨‍👩‍👧‍👦 MY FAMILY  |  🎵 MY MUSIC  |  🎮 MY GAMES
```

Each student picks 1–2 sections and creates a post with drawings, photos, or text.

---

## Instructions for Kids

**EN:**
1. Your pod has a Padlet board with 4 sections: Food, Family, Music, Games
2. Pick one or two sections to post in
3. Create a post with: a drawing, a photo, or write about it
4. Try to include a word in your language!
5. You have 15 minutes — work together!

**עב:**
1. לקבוצה שלכם יש לוח פאדלט עם 4 חלקים: אוכל, משפחה, מוזיקה, משחקים
2. בחרו חלק אחד או שניים לפרסם בו
3. צרו פוסט עם: ציור, תמונה, או כתבו על זה
4. נסו לכלול מילה בשפה שלכם!
5. יש לכם 15 דקות — עבדו יחד!

**عر:**
1. مجموعتكم لديها لوحة بادلت بأربعة أقسام: طعام، عائلة، موسيقى، ألعاب
2. اختاروا قسماً أو اثنين للنشر فيهما
3. أنشئوا منشوراً بـ: رسمة، صورة، أو اكتبوا عنه
4. حاولوا إضافة كلمة بلغتكم!
5. لديكم ١٥ دقيقة — اعملوا معاً!

---

## Sharing (Last 5 Minutes of Activity Block)
- Return to main room
- Teacher screen-shares the Padlet board (scroll through sections)
- Teacher highlights similarities: "Both groups posted about rice!" / "You both like football!"
- Celebrate the posts — this board stays live as the Session 2 portfolio
