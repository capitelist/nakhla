# Activity: My Hello Card — Collaborative Jamboard

## Overview
Each pod creates a shared digital card introducing themselves using Google Jamboard (or Google Slides as alternative).

**Duration:** 15 minutes
**Group:** Full group, working in pod pairs

---

## Setup (Teacher, Before Session)
1. Create one Jamboard per pod (or one Google Slides file per pod)
2. Set sharing to "Anyone with the link can edit"
3. Prepare each board with a template:

### Template Layout (One Frame/Slide)
```
┌─────────────────────────────────────────┐
│           🌴 OUR HELLO CARD 🌴          │
├──────────────────┬──────────────────────┤
│   [Name 1]       │   [Name 2]           │
│   🏠 I live in:  │   🏠 I live in:      │
│   👋 Hello =     │   👋 Hello =         │
│   ⭐ I like:     │   ⭐ I like:         │
├──────────────────┼──────────────────────┤
│   [Name 3]       │   [Name 4]           │
│   🏠 I live in:  │   🏠 I live in:      │
│   👋 Hello =     │   👋 Hello =         │
│   ⭐ I like:     │   ⭐ I like:         │
└──────────────────┴──────────────────────┘
│  Words we learned together:              │
│  _______ _______ _______ _______        │
└──────────────────────────────────────────┘
```

---

## Instructions for Kids

**EN:**
1. Find your name on the card
2. Fill in your section:
   - Where you live
   - How to say "hello" in your language
   - One thing you like (draw it or write it!)
3. At the bottom, write the new words you learned from your partners
4. You can draw, add stickers, or use colors!

**עב:**
1. מצאו את השם שלכם בכרטיס
2. מלאו את החלק שלכם:
   - איפה אתם גרים
   - איך אומרים "שלום" בשפה שלכם
   - דבר אחד שאתם אוהבים (ציירו או כתבו!)
3. בתחתית, כתבו את המילים החדשות שלמדתם מהחברים שלכם
4. אפשר לצייר, להוסיף מדבקות, או להשתמש בצבעים!

**عر:**
1. ابحثوا عن اسمكم في البطاقة
2. املأوا قسمكم:
   - أين تعيشون
   - كيف تقولون "مرحباً" بلغتكم
   - شيء واحد تحبونه (ارسموه أو اكتبوه!)
3. في الأسفل، اكتبوا الكلمات الجديدة التي تعلمتموها من زملائكم
4. يمكنكم الرسم، إضافة ملصقات، أو استخدام الألوان!

---

## Sharing (Last 5 Minutes)
- Teacher screen-shares each pod's card with the full group
- Each pod gets 30 seconds to show their card
- Teacher highlights: "Look — they learned each other's words!"

---

## Alternative: Google Slides
If Jamboard is unavailable:
1. Create a Google Slides presentation
2. One slide per pod, same template layout
3. Share link with edit access
4. Same activity, same instructions
