# Activity: Friendship Postcard — Collaborative Jamboard

## Overview
Each pod creates a two-sided digital "postcard" celebrating their new friendship.

**Duration:** 15 minutes (within pod pair block, last 15 minutes)
**Group:** Pod pairs on shared Jamboard/Google Slides

---

## Setup (Teacher, Before Session)
1. Create one Jamboard (or 2-slide Google Slides) per pod
2. Set sharing to "Anyone with the link can edit"
3. Prepare with template:

### Template — Frame/Slide 1: "FROM US"
```
┌─────────────────────────────────────────┐
│        🌴 FRIENDSHIP POSTCARD 🌴        │
│           FROM: [Pod Name]               │
├──────────────────┬──────────────────────┤
│  SOMETHING FROM  │  SOMETHING FROM      │
│  OUR CULTURE     │  YOUR CULTURE        │
│                  │                       │
│  (Israeli kids   │  (Emirati kids        │
│   draw here)     │   draw here)          │
│                  │                       │
└──────────────────┴──────────────────────┘
```

### Template — Frame/Slide 2: "OUR MESSAGE"
```
┌─────────────────────────────────────────┐
│         ✉️ A MESSAGE FROM US ✉️          │
│                                          │
│  In English: ________________________    │
│                                          │
│  In Hebrew: _________________________    │
│                                          │
│  In Arabic: _________________________    │
│                                          │
│  Signed: [all 4 names]                   │
└──────────────────────────────────────────┘
```

---

## Instructions for Kids

**EN:**
1. **Side 1 — Draw:** Israeli kids: draw something from YOUR culture. Emirati kids: draw something from YOUR culture. Show each other your worlds!
2. **Side 2 — Write:** Together, write a friendship message in ALL 3 languages. Help each other with the words!
3. Sign all 4 of your names at the bottom
4. Be creative — use colors, stickers, drawings!

**עב:**
1. **צד 1 — ציירו:** ילדים ישראלים: ציירו משהו מהתרבות שלכם. ילדים אמירתיים: ציירו משהו מהתרבות שלכם. הראו אחד לשני את העולמות שלכם!
2. **צד 2 — כתבו:** ביחד, כתבו הודעת חברות ב-3 שפות. עזרו אחד לשני עם המילים!
3. חתמו את כל 4 השמות שלכם בתחתית
4. היו יצירתיים — השתמשו בצבעים, מדבקות, ציורים!

**عر:**
1. **الجانب ١ — ارسموا:** الأطفال الإسرائيليون: ارسموا شيئاً من ثقافتكم. الأطفال الإماراتيون: ارسموا شيئاً من ثقافتكم. أروا بعضكم عوالمكم!
2. **الجانب ٢ — اكتبوا:** معاً، اكتبوا رسالة صداقة بالـ٣ لغات. ساعدوا بعضكم بالكلمات!
3. وقّعوا بأسمائكم الأربعة في الأسفل
4. كونوا مبدعين — استخدموا الألوان والملصقات والرسومات!

---

## Sharing (In Full Group Activity Block)
- Teacher screen-shares each pod's postcard with the full group
- Each pod reads their trilingual message aloud
- Teacher saves all postcards to the Google Sites portfolio
