# Microsoft Flip Prompt — Session 3: Friends Across the Shore

## Topic Title
**Message to My Pen Pal**

## Prompt for Students

> Record a message to your pen pal. Tell them one thing you learned about their culture and one question you still have.

## Instructions for Teacher

Post this prompt as the Topic description in Microsoft Flip:

```
🌴 Nakhla Session 3: Friends Across the Shore

This is your last Flip video! Record a message (30–60 seconds) to your pen pal:
1. Tell them ONE THING you learned about their culture
2. Ask them ONE QUESTION you still have
3. Say goodbye in their language (try it — it's okay if it's not perfect!)

This is your friendship message. Make it count! 💚
```

## Settings
- **Video length:** 30–60 seconds
- **Moderation:** ON (teacher approves before visible)
- **Camera + screen recording:** Both enabled
- **Text responses:** Enabled (for students who prefer writing)

## When to Assign
- Assign in Google Classroom **after Session 3** as the final activity
- Share approved videos with parents alongside the Friendship Certificate and Padlet portfolio
- These videos become a lasting memory of the program
