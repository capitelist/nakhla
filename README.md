# Nakhla 🌴

**Nakhla** (نخلة / נַקְלָה — "date palm") is a 3-session cultural exchange program connecting Israeli and Emirati children ages 8–12 through live video, quizzes, collaborative boards, and async video pen pals — fully trilingual in Hebrew, Arabic, and English.

The palm tree grows in both lands. So does friendship.

---

## The 3-Session Arc

| Session | Title | Theme |
|---------|-------|-------|
| 1 | Hello, Neighbor | Greetings, introductions, where we live |
| 2 | Our World | Food, family, music, games |
| 3 | Friends Across the Shore | Conversation, art, goodbye ceremony |

Each session is 60 minutes: Full Group → Pod Pairs (breakout rooms) → Activity → Wrap-Up.

---

## Tool Stack

| Purpose | Tool |
|---------|------|
| Curriculum / LMS | Google Classroom |
| Quizzes | Blooket (blooket.com) |
| Cultural exchange boards | Padlet |
| Live sessions | Zoom + Zoom Interpretation (Hebrew / Arabic / English) |
| Async pen pals / video messages | Microsoft Flip (flip.com) |
| Cultural videos | YouTube unlisted playlists |
| Translation | DeepL (tools/translate.html) |
| Parent comms | WhatsApp |

All tools are free tier.
